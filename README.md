
# Quiz Interativo com JavaScript

Projeto de um quiz dinâmico desenvolvido com **HTML, CSS e JavaScript puro**, criado para praticar manipulação de DOM, eventos, arrays, objetos e lógica de programação.

## Sobre o Projeto

Este quiz gera automaticamente perguntas e respostas a partir de um array de objetos em JavaScript.

Cada pergunta possui alternativas de múltipla escolha, e ao selecionar uma resposta:

- o sistema verifica se está correta
- contabiliza os acertos em tempo real
- atualiza o placar na tela

## Funcionalidades

- Geração dinâmica de perguntas
- Uso de `<template>` para reutilização de estrutura HTML
- Respostas com `input radio`
- Correção automática ao selecionar opção
- Contador de acertos atualizado em tempo real
- Código organizado e comentado

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Conceitos Praticados

- Arrays e Objetos
- `for...of`
- `querySelector()`
- `cloneNode()`
- `appendChild()`
- `remove()`
- `Set()`
- Eventos (`onchange`)
- Manipulação de DOM
- Lógica condicional
